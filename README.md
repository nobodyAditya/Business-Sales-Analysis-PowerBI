# Business Sales Analysis using Excel & Power BI

## Project Overview

This project analyzes business sales performance using Microsoft Excel and Power BI. The objective was to transform raw sales data into meaningful business insights through data preparation, KPI analysis, DAX calculations, and interactive dashboard development.

## Objectives

- Analyze overall sales and profitability
- Identify top-performing regions and categories
- Evaluate product-level performance
- Track monthly sales trends
- Generate actionable business recommendations

## Tools Used

- Microsoft Excel
- Power BI
- DAX Measures
- Pivot Tables
- Data Visualization

## Key KPIs

| KPI | Value |
|------|--------|
| Total Sales | $2.30M |
| Total Profit | $286.40K |
| Profit Margin | 12.03% |

## Dashboard Features

- Monthly Sales Trend Analysis
- Profit by Region
- Sales by Category
- Top Products Analysis
- Sales Category Segmentation
- Interactive Filters (Date, Region, Category)

## Key Insights

- West region generated the highest sales contribution.
- Technology emerged as the highest-performing category.
- Sales increased significantly during the final months of the year.
- Canon imageCLASS was the top-performing product.
- Phones and Chairs were the leading sub-categories.

## Business Recommendations

1. Increase investment in the West region.
2. Expand Technology product offerings.
3. Prepare inventory for peak sales periods.
4. Focus marketing on high-performing products.
5. Improve performance in lower-contributing regions.
